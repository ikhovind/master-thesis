\section{CTaps Overview}
\subsection{Choice of implementation language}
Since a key desire when designing the initial task for
the master thesis was to reach as many developers as possible,
especially lower-level developers who normally work with sockets,
it was decided that the initial implementation should be done in C.
C forms the backbone of many modern programming libraries, ensuring
that a C implementation of TAPS could reach as widely as possible.
